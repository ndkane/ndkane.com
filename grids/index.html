A Brief Intermission

Before I go any further and explain what is meant by a grid, I’d like to draw your attention to this monstrosity:

h1 36px
h2 24px
h3 16px

I attempted to teach myself how to create a table so that I could demonstrate vertical rhythm, however, as you can see, it didn’t quite work. I wanted to create a nicely spaced table with background colour that also had spaces between the cells. I have tried the following:

  • Realising that because I reset the CSS the default settings won't apply, therefore explaining why it isn’t quite formatting properly
  • Adding padding and margins into the table element, however that didn’t work for reasons I don’t quite understand.
  • Asking Stackoverflow and the Internets, but I don’t think I'm asking the right question perhaps.

After asking Aral, I realised that I was looking at the wrong elements altogether. Rather than trying to change the properties of the <table> element, I should have been looking at the elements that control the rows and cells of a table — the <td> and <tr> tags. It was there that I sorted out the margins and padding, which was the problem all along. There’s still an issue with the top margin/padding, but I reckon that might have to do with the padding on <p>!
